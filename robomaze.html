Modes

Clock This mode shows the time. Pressing BACK changes mode. Date This mode shows the date. It returns to time after a few seconds. Stopwatch This is a second accurate counter. UP is lap and reset, while DOWN is start and stop. The stopwatch will stop running sometime after one year, and will loop every 960 hours. This keeps it day synchronized over long time scales. It still counts when the watch loses power. Uninstallation clears the persistant memory, and so resets the stopwatch. Calculator The UP and DOWN buttons increase or decrese the unit digit, and a long press does a multiply or divide by ten. Long pressing SELECT sets up exponents, with long presses for the tens digit of the exponent. A short press of SELECT, sets up function entry.[in dev]. Game The UP and DOWN buttons, when not in stopwatch or calculator mode, select game A/B. When in game mode, the UP and DOWN buttons, select the anticlockwise and clockwise player motion preference, as indicated by the dot next to the score. H and L are displayed when the hiscore and level are displayed. Pressing BACK in game mode goes back to time, and plays the game passively in the background. [in dev]. The SELECT button enters the console mode. The console has a different context based on the mode.[in dev].
